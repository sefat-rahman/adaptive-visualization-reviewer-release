# Adaptive Visualization Static Demo

This folder is a GitHub Pages export of the currently saved precomputed
dashboard states/counties. It does not run Flask or Python in the browser.

- Open `index.html` through GitHub Pages.
- Snapshot routes: `852`
- Generated: `2026-05-28T22:06:17.570199+00:00`

If a state, county, threshold, or analysis is not bundled here, clone the
repository, run `python run_dashboard.py` from the parent
`adaptive_visualization/` folder to generate additional data, then rerun
`python export_static_pages.py`.
